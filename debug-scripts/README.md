# Debug Scripts

This folder contains temporary debugging and testing scripts used during development.

## Scripts

- `check-parent-color.js` - Timer color inheritance debugging
- `debug-timer-color.js` - Timer text color debugging
- `fix-timer-color.js` - Timer color fix testing
- `force-clear-cache.js` - Service worker cache clearing utility
- `clear-cache-v2.js` - Alternative cache clearing script
- `test-streak-logic.js` - Workout streak calculation testing

## Usage

These scripts are kept for reference but are not part of the main application.
They can be run in the browser console or via Node.js for debugging purposes.

## Note

These scripts may be outdated and might not work with the current codebase.
They are preserved for historical reference and debugging techniques.
