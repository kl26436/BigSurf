What changed vs Option B

  • Rest timer is teal/primary, not yellow. Matches the app accent. Still prominent at the top.
  • Equipment is a dedicated card with a "Change" button. One tap to switch mid-workout. Auto-picks most-recently-used.
  • Top progress pills replace the dots — each exercise is a named pill you can tap to jump to.
  • "All" button in the bottom-left opens a drawer showing the full workout. Tap any exercise to jump.
  • Superset mode shows both paired exercises stacked in one view with the active half highlighted. Pair via the drawer → "Superset" button.
  • Bodyweight exercises (Dips example) get the same BW banner from MD 2 — no special UI difference.
  • Auto-fill from last session (KEPT from current app) — every set cell is pre-populated with what you did last time (dashed muted border, muted text color). Tap a cell to edit → border goes solid, text brightens. Tap ✓ to confirm the autofilled values verbatim. Fastest possible logging when you're repeating numbers.
  • Finish button only becomes "Finish workout" on the last exercise; otherwise it's "Next exercise".

How the workflow chains together

Start workout from template → Exercise 1 loads (pills at top, equipment card, last session, set table) → log sets (rest timer auto-appears after each ✓) → Next exercise button OR tap a pill OR "All" → sheet to jump → Finish button on the last exercise → Summary modal with PRs + stats + save.

Supersets: open "All" drawer, tap "Superset", select 2+ exercises, done. Those exercises now render in the grouped superset view when you reach them. Alternatively: long-press an exercise in the drawer to toggle "Link with next" quickly.
