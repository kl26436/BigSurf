Create Workout — Redesign

What's wrong with the current form and how the polished version fixes it. Matches the .hero-card / .row-card / drag-handle patterns used elsewhere in the redesigned app.

Problems with current form

  • Mystery checkbox on each exercise row — no label, no clear purpose. Cut it.
  • Up/down arrows for reordering — wrong pattern on mobile. Use drag handles like the rest of the app.
  • Two heavy icon buttons (edit pencil + trash) competing for space. Consolidate into one overflow menu, and tap the row to edit.
  • Category is a dropdown — feels legal. Use the same colored category chips as everywhere else.
  • Day chips are rectangles with inconsistent tap targets. Use uniform circle chips in a 7-column grid.
  • Asterisk-required labels + oversized input rows — nothing else in the app uses this pattern. Use lighter uppercase field labels.
  • No Save button visible without scrolling. Pin it as a sticky footer with a top-right quick-save too.
  • No estimate preview. Show total sets / reps / est duration so you know what you're building before saving.
