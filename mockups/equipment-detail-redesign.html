B — Identity sheet, derived name
Brand/Line/Function/Type are the source of truth; the display name composes itself (short form for headers, full form for search/matching). Optional nickname replaces the redundant Name input. Backdrop uses the Phase 0 blur. Save patches only the hero — no scroll-to-top.
Migration note: existing equipment already stores name + brand + line + function separately, and most names are the concatenation — so on load, if name === autoName(brand, line, function) (or fuzzy-matches it), treat nickname as unset; otherwise the existing custom name becomes the nickname. Zero data loss, no schema change required (nickname can live in the existing name field; "derived" is a render-time rule). The header/title truncation problem ("Arsenal Strength Reloaded — ISO In… + Done") disappears because headers use the short function name.
